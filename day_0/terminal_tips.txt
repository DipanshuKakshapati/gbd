To clone a repo, create and push your branch and commit your changes in your branch:

1. git clone {repo url}
2. cd {repo}
3. git chekcout -b {new branch}
4. git push origin {branch name}
5. git add {file name}
6. git commit -m '{commit message}'
7. git push origin {branch name}
